Что такое МузМульт?

МузМульт — это виртуальный музей мультфильмов СССР, созданный с любовью к советской анимации. Наша коллекция охватывает работы трёх легендарных студий: московского «Союзмультфильма», киевского «Киевнаучфильма» и телевизионного объединения «Экран».


Советская анимация — это уникальное явление мировой культуры. Мультфильмы, созданные в период с 1940-х по 1990-е годы, отличались высочайшим художественным уровнем, глубоким содержанием и искренностью. МузМульт создан для того, чтобы эта эпоха никогда не была забыта.

Советские мультфильмы — это не просто детские истории. Это целая философия добра, честности и человечности, упакованная в рисованные кадры. Наша миссия — сохранить эту философию и передать её следующим поколениям.

— МузМульт, 2024

История создания музея

Этап 1 — Идея

Концепция и исследование

Идея создать виртуальный музей советской анимации возникла из желания сохранить и популяризировать великое наследие советского мультипликационного искусства. Был проведён отбор студий и произведений.

Этап 2 — Контент

Сбор материалов

Изучение истории каждой студии, подбор экспонатов и написание описаний для произведений. Формирование трёх залов: Киевнаучфильм, Союзмультфильм и ТО «Экран».

Этап 3 — Разработка

Создание сайта

Вёрстка всех страниц, разработка интерактивных элементов — слайдер, вкладки залов, адаптивный дизайн для мобильных устройств и планшетов.

Этап 4 — Запуск

Публикация на GitHub Pages

Тестирование на разных устройствах, финальная проверка контента и публикация музея на GitHub Pages. МузМульт открыт для всех бесплатно, круглосуточно.

⚡

Технологии

Сайт создан на чистых веб-технологиях, без сторонних фреймворков. Весь код написан вручную с вниманием к деталям и производительности.

🟧 HTML5 🟦 CSS3 🟨 JavaScript 📱 Responsive Design 🎨 CSS Custom Properties ♿ ARIA Accessibility 🚀 GitHub Pages
✨

Возможности сайта

🎠 Слайдер — на главной странице изображения переключаются с плавной анимацией, поддержкой свайпа и клавиатуры.


🎬 Залы по студиям — три вкладки с экспонатами, описаниями и возможностью просмотра изображений в полный экран.


🖥️ Адаптивный дизайн — сайт корректно отображается на любом устройстве: от смартфона до широкоформатного монитора.


⌨️ Доступность — поддержка клавиатурной навигации и ARIA-разметка для вспомогательных технологий.
